# EnlightenedC Language Features

This document outlines the features of EnlightenedC. To best understand this language, you should have
a working knowledge of C, as well as an object-oriented language like C++ or Python. There is also
a [guide](https://holyc-lang.com/docs/intro) to introduce you to the main features of HolyC, and then you
can come back here to see the differences.
